# Pairing Slackbot
 
## Prerequisites
1. Install Docker
2. Install AWS Sam
3. Install Pyhton 3.7  (_make sure this is what is being used at runtime (python --version)_)


## How To Run This
1. pip install -r requirements.txt 
2. run myob-auth l & login with your credientials
4. run sam build
5. run sam deploy
